# Cryptography Toolkit Web Application

## Description
This project is a web-based implementation of basic cryptographic techniques and key exchange mechanisms. It demonstrates the working of classical and modern cryptography algorithms using an interactive interface.

## Objectives
- To understand basic encryption and decryption techniques
- To implement substitution and transposition ciphers
- To demonstrate secure key exchange using Diffie-Hellman algorithm

## Topics Covered
1. Caesar Cipher (Substitution Cipher)
2. Rail Fence Cipher (Transposition Cipher)
3. Diffie-Hellman Key Exchange (Asymmetric Cryptography)

## Features
- Encryption and Decryption for Caesar Cipher
- Encryption and Decryption for Rail Fence Cipher
- Diffie-Hellman Key Generation with Prime Number Validation
- User-friendly interface using HTML, CSS, and JavaScript
